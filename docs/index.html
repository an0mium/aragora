aagora

A society of heterogeneous AI agents that discuss, critique, improve each other's responses, and learn from successful patterns.

PyPI v0.1.0 GitHub MIT License
pip install aagora

Heterogeneous Agents

Mix Claude, Codex, Gemini, Grok, Qwen, Deepseek, and local models. 7 major AI labs in one debate.

Structured Debate

Propose, Critique, Revise loop with configurable rounds and multiple consensus mechanisms.

Self-Improvement

SQLite-based pattern store learns from successful critiques. Prompts evolve over time.

One Command

Simple CLI interface. Multiple agents working behind the scenes to produce better answers.

$ aagora ask "Design a rate limiter for 1M req/sec" --agents codex,claude [Round 1] codex_proposer: Proposing solution... [Round 1] claude_critic: Analyzing proposal... Issues found: Race condition in distributed counter [Round 2] codex_proposer: Revising based on critique... [Round 2] claude_critic: Re-evaluating... Consensus reached (confidence: 87%)
