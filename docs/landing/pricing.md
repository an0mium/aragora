## BYOK Model

Aragora uses a **Bring Your Own Key** model. You use your own LLM provider API keys (Anthropic, OpenAI, Google, etc.). Aragora does not mark up LLM costs.

- **Cost per debate:** $0.05-$0.30 depending on model mix (billed by your LLM providers)
- **Demo mode:** Free, no API keys needed
- **No hidden costs:** Infrastructure is included in the subscription

## FAQ

### Can I try before buying?
Yes. The Free tier includes 10 debates/month forever. Demo mode requires no API keys at all.

### Do I need API keys?
Demo mode works without any API keys. For production use, bring your own keys from Anthropic, OpenAI, Google, or other supported providers. Aragora does not charge for LLM usage.

### What happens if I exceed my debate limit?
Upgrade to Pro for unlimited debates.

### Is my data used for training?
Never. Your inputs and outputs are never used to train AI models.

### Can I cancel anytime?
Yes. Monthly plans can be canceled anytime.

### Do you offer discounts?
- **Startups:** 50% off first year (requires accelerator affiliation)
- **Non-profits:** 50% off all plans
- **Education:** Free Pro tier for verified .edu emails
