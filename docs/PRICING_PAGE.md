## Plan Details

For individual developers and teams evaluating Aragora.

The full `aragora-debate` standalone package is MIT-licensed with no usage limits. The Free tier applies to the hosted platform and full `aragora` package with API access.

**Includes:**
- 100 debates per month with up to 3 agents each
- 10 AI code reviews per month
- 5 Gauntlet adversarial stress tests per month
- Decision receipts in Markdown format
- Python and TypeScript SDKs
- Demo mode with no API keys required
- All 43 agent types (bring your own API keys)
- BYOK model: Aragora never marks up LLM provider costs

```bash
pip install aragora-debate
```

## Frequently Asked Questions

### Is the open source version limited?

No. The `aragora-debate` package (the standalone debate engine) is MIT-licensed with no usage limits. It includes the full debate engine, all consensus methods, decision receipts, evidence quality analysis, hollow consensus detection, and the event system. The Free tier limits apply to the hosted platform and full `aragora` package.

### Do I need API keys from LLM providers?

Yes. Aragora orchestrates models from Anthropic, OpenAI, Google, xAI, Mistral, and others. You bring your own API keys and pay providers directly. Demo mode (`--demo` or `create_agent("mock", ...)`) works without keys for evaluation.

### What counts as a "debate"?

One debate is a single run of the adversarial debate engine on one question, regardless of the number of rounds. A 3-round debate with 5 agents counts as 1 debate.

### Can I try before buying?

Yes. Install the open source package, run a debate with mock agents, and evaluate the full API surface immediately:

```bash
pip install aragora-debate
python -c "
import asyncio
from aragora_debate import Debate, create_agent

async def demo():
    debate = Debate('Should we adopt microservices?')
    debate.add_agent(create_agent('mock', name='analyst'))
    debate.add_agent(create_agent('mock', name='critic'))
    result = await debate.run()
    print(result.receipt.to_markdown())

asyncio.run(demo())
"
```

Or try the full platform CLI:

```bash
pip install aragora
aragora review --demo
```

### What is included in "managed deployment"?

We run a dedicated Aragora instance on your cloud account (AWS, GCP, or Azure). You get automatic updates, monitoring, and backup without managing infrastructure. Your data stays in your account. No phone-home telemetry.

### How does EU AI Act compliance work?

Aragora generates Article 12 (Record-Keeping), Article 13 (Transparency), and Article 14 (Human Oversight) compliance artifact bundles from your decision receipts. These are audit-ready documents for conformity assessment under the EU AI Act (enforcement begins August 2, 2026). Available on Pro and Enterprise tiers.

### Can I self-host?

Yes. Enterprise customers can deploy on their own infrastructure with Docker Compose, Kubernetes (Helm chart included), or Terraform. Offline and air-gapped deployments are supported with `aragora serve --offline`.

### How does pricing work for teams?

Pro is priced per seat at $49/month. A "seat" is a user account that can run debates, code reviews, and access the API. Read-only access (viewing receipts, dashboards) does not require a seat.
